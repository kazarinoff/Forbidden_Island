Switch-Case

Written by Zach

An interesting coding bit I’ve recently discovered is the “SWITCH, CASE” logic.

_

This is a bit of logic that is used in substitution of specific If/Else statements where the conditional has a few variations checking on the same variable. So for instance if you want to display something if “x is greater than 5” and display something else if “x is 0” and display something else if “x is less than 0”. This is the perfect syntax for that.

Note examples in this post will be in javascipt, but switch is also used in Ruby(‘case/when’), Java, C#, and many more. Notably Python does not have this functionality- see [this thread] (https://stackoverflow.com/questions/1429505/does-python-have-an-equivalent-to-switch) for some alternatives. TBH I’m not really clear on why it hasn’t been implemented.

Lets take a look at an example:

function stoplight(color){
    switch(color){
        case (green):
            console.log("GOGOGO!");
            break;
        case (yellow):
            console.log("maybe slowdown");
            break;
        case (red):
            console.log("STOP!!!");
            break;
    }
}

Here our function takes in a stoplight color and outputs the appropriate backseat driver response.

That’s the basics.
